# Design Sales Funnel Structure

## Purpose
Architect high-converting sales funnels that guide prospects from cold traffic to purchase/booking, optimized for each funnel type and business model.

## When to Use
- Client needs complete funnel strategy
- Launching new offer or service
- Optimizing underperforming funnel
- Transitioning from one funnel type to another

## Core Funnel Anatomy

Every winning funnel has:
1. **Traffic Source** - Where people come from
2. **Landing Page** - First impression/opt-in
3. **Value Delivery** - Build trust and authority
4. **Offer Presentation** - Make the pitch
5. **Application/Purchase** - Conversion point
6. **Follow-Up** - Nurture and close

## Complete Funnel Type Reference

### 1. VSL (Video Sales Letter) Funnel

**Best For:**
- High-ticket B2B services ($3K-$50K+)
- Complex offers needing explanation
- Coaching/consulting/agency services
- When you need to build authority quickly

**Structure:**
```
COLD TRAFFIC (FB/IG/LinkedIn Ads, Cold Email)
    ↓
LANDING PAGE (Headline + VSL)
    ↓
VIDEO SALES LETTER (3-5 minutes)
    ↓
CALENDAR BOOKING (Direct to Calendly)
    ↓
SALES CALL (Close the deal)
```

**Landing Page Components:**
- **Headline:** Big promise or question (H1)
- **Subheadline:** Clarify who it's for (H2)
- **Video Player:** Autoplay VSL
- **CTA Button:** Below video (Book a Call)
- **Trust Indicators:** Client logos, testimonials
- **Footer:** Privacy, terms, contact

**VSL Script Structure:**
1. Hook (10 sec) - Pattern interrupt
2. Problem (30 sec) - Agitate pain
3. Solution intro (20 sec) - Your mechanism
4. Credibility (45 sec) - Proof and authority
5. Deep dive (2-3 min) - How it works
6. Offer + CTA (30 sec) - Book the call

**Traffic Strategy:**
- Facebook/Instagram ads to cold audience
- LinkedIn ads for B2B
- YouTube pre-roll
- Cold email linking to VSL page

**Conversion Benchmarks:**
- Landing page → Video view: 60-80%
- Video completion: 40-60%
- Video → Calendar: 5-15%
- Call show rate: 60-80%
- Call → Close: 20-40%

**When to Use:**
✅ Selling high-ticket services
✅ B2B offers
✅ Need to educate market
✅ Founder is good on camera

❌ E-commerce/low-ticket
❌ Impulse purchases
❌ No founder camera presence

### 2. Webinar Funnel

**Best For:**
- Mid to high-ticket offers ($997-$10K)
- Teaching-based businesses
- Group coaching/programs
- Software with education component

**Structure:**
```
COLD TRAFFIC
    ↓
REGISTRATION PAGE (Opt-in for webinar)
    ↓
EMAIL SEQUENCE (Reminder emails)
    ↓
WEBINAR PAGE (Live or automated)
    ↓
    ├→ PITCH + DIRECT PURCHASE
    └→ APPLICATION + SALES CALL
    ↓
FOLLOW-UP SEQUENCE (Replay, urgency)
```

**Registration Page Components:**
- Headline: "Free Training: [Specific Outcome]"
- Bullet points: What they'll learn
- Host credibility section
- Date/time options
- Simple opt-in form (Name + Email)
- Social proof

**Webinar Structure (60-90 min):**
1. **Intro (5 min):**
   - Welcome and housekeeping
   - Who you are (briefly)
   - What they'll learn
   - Rules for engagement

2. **Content (30-40 min):**
   - The "What" and "Why"
   - 3 main teaching points
   - Each with: Story → Teaching → Application
   - High value, not pitch
   - Tease the "How" (your offer)

3. **Transition (5 min):**
   - "Now I want to show you HOW to implement this"
   - "I've created a system/program that does X"
   - Permission to pitch

4. **Pitch (15-20 min):**
   - What you're offering
   - Who it's for (and not for)
   - What's included
   - How it works
   - Proof (case studies)
   - Price reveal
   - Bonuses
   - Guarantee
   - Scarcity (limited spots/deadline)
   - CTA (link in chat/apply now)

5. **Q&A (10-15 min):**
   - Answer objections
   - More social proof
   - Final CTA

**Email Sequence:**
- Day 7: Registration confirmation + calendar add
- Day 6: Reminder + pre-frame value
- Day 3: Reminder + testimonial
- Day 1: Final reminder + reason to attend
- 1hr before: Last chance reminder
- Day after: Replay available (24-48hr window)
- Day 2-3: Replay ending soon + final CTA

**Conversion Benchmarks:**
- Ad → Registration: 20-40%
- Registration → Attendance: 20-40%
- Attendance → Purchase: 2-8%
- Overall funnel ROI: 2-5X

**When to Use:**
✅ Educational offers
✅ Want to build authority
✅ Mid-ticket pricing
✅ Can present live/recorded

❌ Need quick conversions
❌ Very high-ticket ($50K+)
❌ Simple/obvious offers

### 3. Challenge Funnel

**Best For:**
- Fitness/health/transformation
- Skills-based learning
- Building engaged community
- Creating "believer" customers

**Structure:**
```
COLD TRAFFIC
    ↓
REGISTRATION PAGE (Join the challenge)
    ↓
PAYMENT ($27-$97 to join)
    ↓
DAILY CONTENT + COMMUNITY (5-7 days)
    ↓
PITCH (Mid-way + end)
    ↓
UPSELL TO MAIN OFFER
```

**Challenge Structure (5-Day Example):**

**Pre-Challenge:**
- Welcome email with calendar
- Facebook group/Slack invite
- Set expectations
- Quick win homework

**Day 1: Foundation**
- Training: Core concept introduction
- Action: Simple implementation task
- Community: Share results
- Seed the offer: Mention what's coming

**Day 2: Skill Building**
- Training: Dive deeper on method
- Action: More advanced implementation
- Community: Celebrate wins
- Seed: Show advanced results

**Day 3: Transformation**
- Training: See results from action
- Action: Full implementation
- First pitch: Introduce main offer
- Community: Testimonials from past challenges

**Day 4: Optimization**
- Training: Refine the approach
- Action: Perfect the system
- Pitch: Detailed offer presentation
- Community: Q&A about offer

**Day 5: Next Level**
- Training: What's beyond the challenge
- Pitch: Final offer push with bonuses
- Deadline: Cart closes tonight
- Community: Celebrate all wins

**Conversion Benchmarks:**
- Ad → Registration: 15-30%
- Registration → Payment: 5-15%
- Payment → Show up: 60-80%
- Attendee → Upsell: 15-30%

**When to Use:**
✅ Transformation-based offers
✅ Want highly qualified buyers
✅ Can deliver daily content
✅ Have community infrastructure

❌ B2B services
❌ Can't support live engagement
❌ Very high-ticket

### 4. Tripwire Funnel

**Best For:**
- E-commerce and info products
- Building buyer lists quickly
- Frontend → Backend strategy
- Recovering ad costs immediately

**Structure:**
```
COLD TRAFFIC
    ↓
LANDING PAGE (Irresistible low-ticket offer)
    ↓
CHECKOUT ($7-$47 product)
    ↓
ORDER BUMP (+$10-$30)
    ↓
OTO1: One-Time Offer (+$97-$297)
    ↓
OTO2: Second upsell (+$297-$997)
    ↓
THANK YOU PAGE (Delivery + next steps)
    ↓
EMAIL SEQUENCE (Main offer pitch)
```

**Tripwire Landing Page:**
- Headline: Specific outcome + Price shock
  - "Get [Specific Result] For Just $7"
- Hero image/video: Show the product
- 5-8 bullet points: What they get
- Social proof: Reviews/testimonials
- Guarantee: Risk reversal
- Scarcity: Limited time/quantity
- CTA button: "Add to Cart" or "Get Instant Access"

**Checkout Page:**
- Order summary
- **Order Bump** (checkbox):
  - "Add [complementary product] for just $19"
  - Show value stack
  - Make it a no-brainer

**OTO1 (Upsell Page):**
- "WAIT! Special One-Time Offer"
- More complete version of what they bought
- Time-sensitive (this page only)
- 2 buttons: "Yes, Add This!" / "No Thanks"

**OTO2 (Second Upsell):**
- Even higher-ticket related offer
- Done-for-you or full system
- Again, time-sensitive
- 2 buttons: Accept / Decline

**Thank You Page:**
- Confirm purchase
- Access instructions
- Set expectations
- Tease email sequence
- Optional: Survey for segmentation

**Email Sequence Post-Purchase:**
- Email 1 (Immediate): Welcome + access
- Email 2 (Day 1): Quick win + implementation
- Email 3 (Day 2): Value + case study
- Email 4 (Day 3): Address objection
- Email 5 (Day 4): Main offer pitch
- Email 6 (Day 5): Scarcity + final push

**Conversion Benchmarks:**
- Landing → Purchase: 3-8%
- Order bump take rate: 20-40%
- OTO1 take rate: 15-30%
- OTO2 take rate: 5-15%
- Back-end offer: 5-15%

**When to Use:**
✅ Have info product or physical product
✅ Want to build buyer list
✅ Have backend offer to promote
✅ Can recover ad costs with tripwire

❌ Only selling services
❌ No low-ticket offer to create
❌ No backend to sell

### 5. Application Funnel

**Best For:**
- Very high-ticket ($10K-$100K+)
- Done-for-you services
- Implementation-heavy offers
- Want to pre-qualify leads

**Structure:**
```
COLD TRAFFIC (Qualified sources)
    ↓
LANDING PAGE (Qualify + filter)
    ↓
APPLICATION FORM (Detailed questions)
    ↓
BOOKING PAGE (If qualified)
    ↓
SALES CALL (High-touch close)
```

**Landing Page Strategy:**
- **Disqualify early**: "This is ONLY for..."
- **High barrier**: Detailed application required
- **Social proof**: Only the best clients
- **No price**: Focus on value and fit
- **CTA**: "Apply Now" (not "Book a call")

**Application Form Questions:**

**Section 1: Business Info**
- Company name
- Website URL
- Industry/niche
- Years in business
- Current monthly revenue
- Revenue goal

**Section 2: Problem/Pain**
- What's your biggest challenge right now?
- What have you tried to solve this?
- Why hasn't it worked?
- How long have you had this problem?

**Section 3: Commitment**
- How much can you invest to solve this? ($5K-$10K, $10K-$25K, $25K-$50K, $50K+)
- When do you need this solved by?
- Who else is involved in decision-making?
- On a scale of 1-10, how committed are you to solving this?

**Section 4: Logistics**
- Preferred call time/timezone
- Phone number
- Email
- How did you hear about us?

**Auto-Qualification Logic:**
- Revenue under minimum → Reject (send to email sequence)
- Budget too low → Reject (offer lower-ticket alternative)
- Not decision maker → Reject or require DM on call
- Commitment under 8/10 → Reject or delay
- Meets all criteria → Approve + calendar link

**Thank You Page (Qualified):**
- "Congratulations, you qualify!"
- "Here's what to expect on the call..."
- "Please book your call below"
- Embedded calendar
- Homework: "To prepare, please [task]"

**Thank You Page (Not Qualified):**
- "Thanks for your application"
- "Unfortunately, we're not the right fit because..."
- "However, we have a resource that might help"
- Link to email sequence or lower-ticket offer

**Conversion Benchmarks:**
- Landing → Application start: 15-40%
- Application complete: 40-70%
- Qualified applications: 30-60%
- Qualified → Booked call: 60-80%
- Call → Close: 25-50%

**When to Use:**
✅ High-ticket offers ($10K+)
✅ Need to pre-qualify heavily
✅ Implementation-intensive service
✅ Want only serious prospects

❌ Want high volume of calls
❌ Lower-ticket offers
❌ Quick/simple products

### 6. E-Commerce Product Funnel

**Best For:**
- Physical products
- Digital products (courses, templates)
- Subscription boxes
- Direct-to-consumer brands

**Structure:**
```
COLD TRAFFIC (FB/IG Ads, TikTok)
    ↓
PRODUCT PAGE (Optimized for conversion)
    ↓
CART (Order bumps, upsells)
    ↓
CHECKOUT (Frictionless)
    ↓
POST-PURCHASE UPSELL
    ↓
THANK YOU (Email capture, next purchase)
    ↓
EMAIL SEQUENCES (Retention, repeat)
```

**Product Page Optimization:**

**Above the Fold:**
- High-quality hero image (5-7 images total)
- Product name (H1)
- Star rating + review count
- Price (with sale price crossed out if applicable)
- CTA button: "Add to Cart" / "Buy Now"
- Trust badges: Free shipping, guarantee, secure checkout

**Product Details:**
- 5-8 bullet points of benefits (not features)
- Use cases and applications
- Specs and dimensions
- Materials/ingredients

**Social Proof:**
- Customer photos/videos
- Star ratings
- Written reviews (with photos)
- "X people bought this today"
- "Only X left in stock"

**Product Story:**
- Founder story (optional)
- Why it was created
- Problem it solves
- How it's different

**FAQ Section:**
- Shipping times
- Return policy
- Sizing/compatibility
- Common questions

**Final CTA:**
- Sticky add-to-cart button
- Guarantee restated
- Urgency (limited stock/sale ending)

**Cart Optimization:**
- Show product image + details
- Order bump checkboxes:
  - "Add [complementary product] - Just $X more"
  - Pre-checked or easy checkbox
  - 2-3 max order bumps

**Checkout Page:**
- **Minimize friction:**
  - Guest checkout option
  - Autofill addresses
  - Multiple payment options (CC, PayPal, Apple Pay, etc.)
  - Trust badges repeated
- **Information order:**
  1. Email
  2. Shipping address
  3. Shipping method
  4. Payment info
- **Show cart summary** (right sidebar)
- **Exit popup:** Discount code if abandoning

**Post-Purchase Upsell:**
- "WAIT! Add this to your order"
- Complementary product
- "Just click yes - no need to re-enter payment"
- One-click upsell
- 30% of people will take it

**Thank You Page:**
- Order confirmation
- What to expect (shipping timeline)
- Tracking info (if available)
- "While you wait" → Content/community
- Incentive to refer friends
- Option to join email/SMS list

**Email Sequence:**
- Email 1 (Immediate): Order confirmation
- Email 2 (Day 2): Order shipped + tracking
- Email 3 (Day 5): Delivery soon + how to use
- Email 4 (Day 8): Getting the most out of [product]
- Email 5 (Day 15): Review request
- Email 6 (Day 30): Replenishment reminder or complementary product

**Conversion Benchmarks:**
- Traffic → Product page: 100%
- Product page → Add to cart: 3-8%
- Cart → Purchase: 30-50%
- Overall conversion rate: 1-4%
- Order bump take rate: 20-35%
- Post-purchase upsell: 20-30%

**When to Use:**
✅ Selling physical or digital products
✅ DTC brand
✅ Want high-volume sales
✅ Lower-ticket items ($20-$500)

❌ Selling services
❌ Need consultation/sales call
❌ Very high-ticket

### 7. Lead Magnet → Nurture Funnel

**Best For:**
- Building email list
- Long sales cycle products
- Trust-building required
- Content-driven businesses

**Structure:**
```
COLD TRAFFIC
    ↓
LANDING PAGE (Lead magnet offer)
    ↓
OPT-IN FORM (Email capture)
    ↓
THANK YOU PAGE (Deliver + upsell)
    ↓
EMAIL NURTURE SEQUENCE (5-10 emails)
    ↓
PITCH EMAIL (Soft sell)
    ↓
SALES PAGE or BOOKING
```

**Lead Magnet Landing Page:**
- Headline: "Free [Thing]: [Specific Outcome]"
- Subheadline: Who it's for
- 3-5 bullets: What they'll get/learn
- Visual mockup of lead magnet
- Opt-in form (just name + email)
- No menu/navigation (dedicated landing page)

**High-Converting Lead Magnets:**
- **Checklists**: "10-Point [Process] Checklist"
- **Templates**: "Copy-Paste [Asset] Template"
- **Guides**: "Complete Guide to [Outcome]"
- **Cheat Sheets**: "[Topic] Cheat Sheet"
- **Webinars**: "Free Training: [How to X]"
- **Toolkits**: "[Niche] Starter Kit"
- **Calculators**: "[ROI/Results] Calculator"

**Thank You Page:**
- Confirm opt-in
- "Check your email for [Lead Magnet]"
- **Immediate upsell opportunity:**
  - "Want the shortcut?"
  - Low-ticket offer ($7-$97)
  - "Get instant access for just $X"

**Nurture Email Sequence:**

**Email 1 (Immediate): Delivery**
- Subject: "Your [Lead Magnet] is ready!"
- Deliver the lead magnet
- Set expectations for future emails
- Quick introduction to you/brand

**Email 2 (Day 1): Value**
- Subject: "[Tip] that works every time"
- Provide quick win related to lead magnet
- No pitch
- Build trust

**Email 3 (Day 2): Story**
- Subject: "How I [achieved result]..."
- Share personal story or case study
- Connect to their pain point
- Introduce methodology

**Email 4 (Day 3): Education**
- Subject: "The [mistake] everyone makes"
- Common mistake or misconception
- How to avoid it
- Position your solution

**Email 5 (Day 4): Objection Handling**
- Subject: "[Objection]? Here's what I think"
- Address main objection
- Share proof it's not an issue
- Seed the solution

**Email 6 (Day 5): Soft Pitch**
- Subject: "Ready for [next step]?"
- Recap value provided
- Introduce paid offer
- Show what's possible
- Clear CTA

**Email 7 (Day 6): Case Study**
- Subject: "How [Name] got [Result]"
- Detailed case study
- Their situation → transformation
- Link to offer

**Email 8 (Day 7): Scarcity**
- Subject: "Only [X] spots left"
- Create urgency
- Bonuses or deadline
- Final CTA

**Ongoing Nurture (Weekly):**
- Content emails (80%)
- Pitch emails (20%)
- Stay top of mind
- Build relationship over time

**Conversion Benchmarks:**
- Ad → Landing page: 10-30%
- Landing → Opt-in: 20-50%
- Thank you → Tripwire purchase: 2-8%
- Email sequence → Offer purchase: 1-5%
- Cost per lead: $2-$10

**When to Use:**
✅ Long sales cycle
✅ Need to build trust first
✅ Educational/content business
✅ Want to grow email list

❌ Need fast conversions
❌ Very simple/obvious offer
❌ No ability to nurture leads

## Funnel Selection Framework

### Choose Based On:

**Ticket Price:**
- Under $100: E-commerce funnel or Tripwire
- $100-$997: Lead magnet or Webinar
- $1K-$10K: VSL or Challenge
- $10K+: Application funnel

**Sales Cycle:**
- Impulse (minutes): E-commerce or Tripwire
- Short (days): VSL or Challenge
- Medium (weeks): Webinar or Lead magnet
- Long (months): Lead magnet → Nurture

**Product Type:**
- Physical product: E-commerce
- Digital product: Tripwire or Lead magnet
- Service (low-touch): VSL or Webinar
- Service (high-touch): Application
- Transformation: Challenge

**Your Strengths:**
- Good on camera: VSL or Webinar
- Great writer: Lead magnet → Email
- Implementation focused: Application
- Product creator: E-commerce or Tripwire

- [ ] Multiple payment options

- [ ] Email sequence written

## Technical Implementation

### Required Tools by Funnel:
- **VSL Funnel**: Landing page builder, Video hosting, Calendar tool
- **Webinar Funnel**: Email platform, Webinar software, Payment processor
- **Challenge Funnel**: Email platform, Community platform, Course platform
- **Tripwire Funnel**: E-commerce platform, Upsell/bump software, Email platform
- **Application Funnel**: Form builder, Scheduling tool, CRM
- **E-commerce Funnel**: Shopify/WooCommerce, Email/SMS platform, Reviews app
- **Lead Magnet Funnel**: Landing page builder, Email platform, Lead magnet delivery

- **Landing Pages**: ClickFunnels, Unbounce, Carrd, Framer, Webflow

- **Webinar**: Demio, WebinarJam, EverWebinar

- **Calendar**: Calendly, Cal.com, TidyCal
- **Forms**: Typeform, Jotform, Google Forms
- **Payment**: Stripe, PayPal, Gumroad

## Funnel Build Process

### Phase 1: Strategy (Week 1)
1. Define offer and price point
2. Identify target audience ICP
3. Map customer journey
4. Choose funnel type
5. Write core messaging
6. Outline all pages/emails

### Phase 2: Creation (Week 2-3)
1. Write all copy (pages + emails)
2. Create lead magnet / VSL / webinar
3. Design all pages
4. Set up technical infrastructure
5. Create graphics and assets

### Phase 3: Implementation (Week 3-4)
1. Build landing pages
2. Connect payment processor
3. Set up email sequences
4. Install tracking pixels
5. Test entire flow (multiple times)

### Phase 4: Launch (Week 4)
1. Soft launch to small audience
2. Monitor for errors
3. Gather initial data
4. Fix any issues
5. Full launch

### Phase 5: Optimization (Ongoing)
1. Analyze funnel metrics weekly
2. A/B test headlines, CTAs, images
3. Improve lowest-converting step
4. Survey customers for feedback
5. Iterate and improve
