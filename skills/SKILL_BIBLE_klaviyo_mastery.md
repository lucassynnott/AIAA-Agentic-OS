## PART 2: STEP-BY-STEP PROCESS

### Phase 1: Klaviyo Account Setup (Day 1)

**Step 1: Account Creation**
1. Go to klaviyo.com
2. Sign up with business email
3. Select ecommerce as business type
4. Choose Shopify integration

**Step 2: Shopify Integration**
1. In Shopify admin, go to Apps
2. Search "Klaviyo"
3. Install Klaviyo app
4. Authorize connection
5. Verify data sync (products, customers, orders)

**Step 3: Domain Authentication**
1. Navigate to Settings > Sending
2. Add your domain
3. Update DNS records as instructed
4. Verify authentication status
5. Set up DMARC policy

**Step 4: Tracking Setup**
1. Confirm on-site tracking is active
2. Verify Shopify integration is pulling data
3. Test form submissions
4. Check order tracking
5. Validate customer profiles

### Phase 2: List Building Setup (Days 2-3)

**Step 1: Create Welcome List**
1. Go to Lists & Segments
2. Create new list: "Newsletter Subscribers"
3. Set up double opt-in (recommended)
4. Configure list settings

**Step 2: Design Pop-up Form**
1. Navigate to Sign-up Forms
2. Create new form
3. Choose pop-up template
4. Customize design elements:
   - Brand colors
   - Logo placement
   - Font selection
   - Mobile optimization

**Step 3: Pop-up Copy Framework**
```
Headline: "Get 10% Off Your First Order"
Subheadline: "Join thousands of happy customers"
CTA Button: "Claim 10% Off"
Disclaimer: "No thanks, I'll pay full price"
```

**Step 4: Targeting Settings**
- Display after 4 seconds on page
- Exit intent trigger
- Show on mobile and desktop
- Don't show again for 5 days after close
- Suppress for existing subscribers

**Step 5: Success Page Setup**
```
Headline: "Check Your Email!"
Body: "Your 10% discount is waiting in your inbox"
CTA: "Start Shopping"
```

## PART 5: DESIGN GUIDELINES

### Email Design Principles

**Mobile-First Approach:**
- 80% of emails are opened on mobile
- Single column layout
- Minimum 44px touch targets
- 14px minimum font size
- Thumb-friendly button placement

**Visual Hierarchy:**
1. Logo/Header (brand recognition)
2. Subject line reinforcement
3. Primary message/offer
4. Call-to-action button
5. Supporting content
6. Footer/unsubscribe

**Color Psychology:**
- Primary brand color for CTAs
- High contrast for readability
- Consistent color scheme
- Accessibility compliance (WCAG 2.1)

### Template Specifications

**Email Width:**
- Maximum: 600px
- Recommended: 500-550px
- Mobile breakpoint: 480px

**Typography:**
- Headline: 24-32px, bold
- Body text: 14-16px, regular
- CTA buttons: 16-18px, bold
- Line height: 1.4-1.6

**Button Design:**
- Minimum height: 44px
- Padding: 12px 24px
- Border radius: 4-8px
- Contrasting color
- Clear, action-oriented text

**Image Guidelines:**
- Product images: 300-500px wide
- Hero images: 500-600px wide
- File size: Under 1MB total
- Alt text for accessibility
- Retina-ready (2x resolution)

### Brand Consistency

**Logo Placement:**
- Header: Left-aligned or centered
- Size: 150-200px wide maximum
- High-resolution PNG/SVG
- Linked to homepage

**Font Selection:**
- Maximum 2 font families
- Web-safe fonts preferred
- Consistent hierarchy
- Brand guideline compliance

**Color Palette:**
- Primary: Brand main color
- Secondary: Accent color
- Neutral: Gray scale
- CTA: High-contrast color
