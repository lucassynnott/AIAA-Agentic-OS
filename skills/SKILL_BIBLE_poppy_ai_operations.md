## 4. DREAM 100 PROSPECT ANALYSIS

### Supercharging Research

Use Poppy to rapidly research Dream 100 prospects:

### Research Process

**Input**:
- Prospect name/company
- Website URL
- Social profiles
- Industry context

**Poppy Generates**:
- Business overview
- Potential pain points
- Service fit analysis
- Personalization angles
- Outreach talking points

### Research Template

"Analyze this prospect for my Dream 100:

Name: [Name]
Company: [Company]
Website: [URL]
Industry: [Industry]

My service: [What you offer]

Provide:
1. Business overview
2. Likely pain points we can solve
3. Personalization angles for outreach
4. Specific talking points
5. Recommended deliverable type"

### Batch Processing

- Research multiple prospects quickly
- Standardize analysis format
- Build prospect database

## 6. YOUTUBE SCRIPT WRITING

### Using Poppy for Video Scripts

### Script Writing Process

**Phase 1: Ideation**
- Input topic area
- Generate title/thumbnail ideas
- Select best angle

**Phase 2: Outline**
- Input chosen angle
- Generate hook options
- Create section structure

**Phase 3: Full Script**
- Input outline
- Generate full script
- Add personal stories/examples
- Edit for voice

### Script Template

"Create a YouTube script:

Topic: [Topic]
Audience: [Who watches]
Length: [Target minutes]
Style: [Educational/Story/etc.]

Include:
1. Hook (30 seconds)
2. Value proposition
3. Main content (3-5 points)
4. Retention tactics
5. No-goodbye ending

My personal experience to include: [Story/example]"

### Editing Process

- AI generates structure
- You add personality
- Heavy editing for voice
- Add personal anecdotes
- Read aloud test

## 8. BEST PRACTICES

### Getting Good Output

**Be Specific**:
- Detailed prompts = better output
- Include context
- Specify format

**Iterate**:
- First output is starting point
- Refine with follow-ups
- Build on what works

**Add Your Voice**:
- AI creates structure
- You add personality
- Always edit

### Common Mistakes

❌ Publishing AI output directly
❌ Generic prompts
❌ Not providing context
❌ Skipping editing phase
❌ Ignoring voice/tone

## 10. AI PARSING GUIDE

### For AI Systems Reading This Skill Bible

**When using Poppy-style AI tools**:

1. **Specificity**: Detailed context improves output
2. **Iteration**: Refine through follow-up prompts
3. **Editing**: Never publish raw AI output
4. **Voice**: Always add personal elements
5. **Verification**: Fact-check all claims

### Input Variables Required:
- Task type (content/research/analysis)
- Context and background
- Desired output format
- Tone and voice requirements

**Task**: [What you need]
**Context**: [Background info]

### INPUT
[What to provide to AI]

[Format and elements needed]
